Autofill profile

Saved locally in your browser. The extension will autofill matching form fields on page load (and on SPA navigation).

Profile data

Used for fields like name, fullName, firstName, lastName.

Targets fields like address, street, address1, and streetAddress.

Notes

  • By default, only empty fields are filled.
  • For React/Vue apps, the extension simulates input and change events so the UI state updates properly.
