Custom fields

Add your own rules for specific fields. The matcher is a simple text or regex that looks at the field's ID, name, or label.

AI generation (optional)

If enabled, the extension can generate answers for certain open-ended questions (e.g. “company values”) using your AI API key. This may send the question text (and nearby context) to the provider.

Available variables: {{company}}, {{question}}, {{context}}.

Notes

  • By default, only empty fields are filled.
  • For React/Vue apps, the extension simulates input and change events so the UI state updates properly.
